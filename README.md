# 📚 Orçamento Pessoal e Planejamento Financeiro com NotebookLM

Projeto desenvolvido como parte do desafio da **DIO**, utilizando o **NotebookLM** como ferramenta de aprendizagem ativa para estudar conceitos introdutórios de finanças pessoais.

---

# 📖 Descrição

Este projeto teve como objetivo explorar o uso do NotebookLM para organizar conhecimentos sobre **Orçamento Pessoal e Planejamento Financeiro**.

Durante o desenvolvimento foram utilizadas fontes oficiais para criar um caderno temático, elaborar perguntas estratégicas, testar diferentes prompts e produzir um material de estudo estruturado.

Ao final, foi desenvolvido um miniguia contendo resumos, glossário financeiro e prompts reutilizáveis para futuras revisões.

---

# 🎯 Objetivos

- Compreender os conceitos fundamentais do orçamento pessoal.
- Aprender a organizar receitas e despesas.
- Entender a importância do planejamento financeiro.
- Conhecer estratégias para reduzir gastos.
- Aprender como criar uma reserva de emergência.
- Desenvolver hábitos financeiros saudáveis.

# 📘 Resumo do Conteúdo

## Orçamento Pessoal

Ferramenta utilizada para controlar receitas e despesas, permitindo acompanhar a situação financeira.

## Planejamento Financeiro

Processo de definição de objetivos financeiros e estratégias para alcançá-los.

## Regra 50-30-20

- 50% para necessidades
- 30% para desejos
- 20% para investimentos e reserva de emergência

## Reserva de Emergência

Valor reservado para cobrir despesas inesperadas.

## Controle de Gastos

O acompanhamento constante das despesas ajuda a identificar desperdícios e manter o equilíbrio financeiro.

# 🔍 Troubleshooting

Durante o desenvolvimento foram observados alguns desafios:

- Prompts muito genéricos geravam respostas superficiais.
- Prompts específicos produziram respostas mais completas.
- Solicitar exemplos práticos melhorou significativamente a qualidade do material.
- Comparar diferentes fontes tornou as respostas mais confiáveis.
- A utilização de fontes oficiais aumentou a consistência das informações.

Esses aprendizados demonstraram a importância da engenharia de prompts para obter melhores resultados com Inteligência Artificial.
